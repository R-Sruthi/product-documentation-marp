# Algorithmic Complexity

We are optimizing our search algorithm to improve documentation retrieval speed.

The time complexity of our new indexing algorithm is:

$$
T(n) = O(n \log n)
$$

Where:
- $n$ is the number of documents.
- $\log n$ represents the tree traversal depth.
